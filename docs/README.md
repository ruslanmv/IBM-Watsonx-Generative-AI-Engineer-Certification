## Video Format

Each video follows this structure:
- **Introduction** (30 seconds): What you'll learn
- **Concept Explanation** (5-10 minutes): Theory with visuals
- **Demo/Example** (5-10 minutes): Real watsonx.ai demonstrations
- **Summary** (1-2 minutes): Key takeaways
- **Next Steps** (30 seconds): What's coming next

## Slide Design Guidelines

- **Simple and Clean**: One concept per slide
- **Visual Focus**: Diagrams over text
- **IBM Branding**: Use IBM color palette (blue: #0072C6)
- **Code Snippets**: Large, readable font with syntax highlighting
- **Animations**: Minimal, only to reveal information progressively

## Script Format

Each script includes:
1. **Verbatim narration** - What to say word-for-word
2. **On-screen actions** - What to demonstrate
3. **Timing markers** - Suggested pacing
4. **B-roll suggestions** - Supporting visuals
5. **Engagement prompts** - Questions to audience, CTAs

## File Naming Convention

- Slides: `module-X-video-Y-slides.html` or `.pdf`
- Scripts: `module-X-video-Y-script.md`
- Assets: `module-X-video-Y-asset-name.png`

Example:
- `module-2-video-3-slides.html`
- `module-2-video-3-script.md`
- `module-2-video-3-architecture-diagram.png`

## Tools Used

- **Slides**: HTML/reveal.js for web-based presentations
- **Screen Recording**: OBS Studio or similar
- **Editing**: DaVinci Resolve, Adobe Premiere, or Final Cut Pro
- **Thumbnails**: Canva or Adobe Photoshop

## YouTube Optimization

- **Video Length**: 10-15 minutes (optimal for retention)
- **Thumbnails**: Include text overlay with key concept
- **Titles**: "[C1000-185] Topic Name - IBM watsonx Certification"
- **Description**: Include timestamps, resources, and links
- **Tags**: IBM, watsonx, certification, generative AI, C1000-185
