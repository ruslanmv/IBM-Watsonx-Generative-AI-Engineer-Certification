\chapter{Glossary of Terms}
\label{app:glossary}

\begin{description}[leftmargin=!, labelwidth=\widthof{\bfseries Temperature}]

\item[Agent] An AI system that can reason, plan, and use tools to accomplish complex tasks.

\item[Attention Mechanism] The component of transformer models that determines which parts of the input to focus on.

\item[Chunking] Breaking large documents into smaller segments for processing or embedding.

\item[Context Window] The maximum amount of text (in tokens) that an LLM can process at once.

\item[Embedding] A numerical vector representation of text that captures semantic meaning.

\item[Fine-Tuning] Adapting a pre-trained model to specific tasks or domains using additional training.

\item[Foundation Model] A large-scale pre-trained model that can be adapted for various tasks.

\item[Hallucination] When an LLM generates plausible but factually incorrect information.

\item[InstructLab] IBM's open-source tool for customizing foundation models with new knowledge.

\item[Knowledge Cutoff] The date beyond which an LLM has no training data.

\item[LangChain] A framework for building applications with language models.

\item[LoRA] Low-Rank Adaptation - A parameter-efficient fine-tuning technique.

\item[Prompt] The input text provided to an LLM to instruct it on a task.

\item[Prompt Engineering] The practice of designing effective prompts to elicit desired LLM outputs.

\item[Quantization] Reducing model precision to decrease size and computational requirements.

\item[RAG] Retrieval-Augmented Generation - Combining LLMs with external knowledge retrieval.

\item[Temperature] A parameter controlling the randomness of LLM outputs (0-1).

\item[Token] The basic unit of text processed by LLMs (roughly 0.75 words).

\item[Top-k] A parameter limiting sampling to the k most likely next tokens.

\item[Top-p] A parameter for nucleus sampling, selecting tokens whose cumulative probability reaches p.

\item[Transformer] The neural network architecture underlying modern LLMs.

\item[Vector Database] A specialized database for storing and querying embedding vectors.

\item[watsonx.ai] IBM's enterprise platform for foundation models and generative AI.

\item[Zero-Shot] Performing a task without providing examples, relying on pre-trained knowledge.

\end{description}
