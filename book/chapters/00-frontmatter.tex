\chapter*{Preface}
\addcontentsline{toc}{chapter}{Preface}

\epigraph{``Build it on the platform, prove it with metrics, then explain it like a teacher.''}{---\ The author's working motto}

\noindent The field of Generative AI is moving fast, and IBM's \textbf{watsonx} platform is where many enterprises will deploy it. This study guide is the companion text for the YouTube and Udemy series of the same name --- all three formats follow the official \textbf{IBM watsonx Generative AI Engineer v1 -- Associate (C1000-185)} exam blueprint.

This first edition was written for the 2026 watsonx.ai catalogue. Every code listing, every model identifier, and every API parameter you read here has been validated against the current \texttt{ibm-watsonx-ai} Python SDK and the public watsonx.ai REST API. Where IBM has retired or renamed a model, the deprecated identifier is called out so you recognise it on the exam without using it in new work.

\section*{Purpose of This Book}

This book is a structured, exam-oriented learning path covering Generative AI concepts, prompt engineering, fine-tuning, Retrieval-Augmented Generation (RAG), deployment, and orchestration on the IBM watsonx platform. Every chapter is mapped 1:1 to a section of the official exam objectives, and every code sample uses the \textbf{current watsonx.ai model catalogue and SDKs} (Granite 3.x, Llama 3.x, Mistral, Granite Guardian, Granite-Embedding, Slate v2, \texttt{ibm-watsonx-ai} Python SDK). Older identifiers such as \texttt{granite-13b-instruct-v2}, \texttt{llama-2-*}, and \texttt{flan-*} are flagged where they appear so you can recognise them on the exam without using them in new work.

\subsection*{Workbook Components}

\begin{itemize}[leftmargin=*]
    \item \textbf{Chapter-End Questions}: Exam-style MCQ and scenario questions at the end of every chapter.
    \item \textbf{Mock Exams}: A 40-question mock (Appendix~B / answer key in Appendix~C) and a longer second practice exam (Appendix~E / answer key in Appendix~F).
    \item \textbf{Glossary}: All the GenAI and watsonx terms used in the book, in Appendix~A.
    \item \textbf{Cheat Sheet}: A one-page exam-day revision card in Appendix~D.
    \item \textbf{Exam Objective Map}: Every blueprint objective (1.1 \dots\ 6.6) cross-referenced to its chapter and page in Appendix~G.
    \item \textbf{Index}: All key terms, watsonx products, and model identifiers indexed at the back of the book.
\end{itemize}
