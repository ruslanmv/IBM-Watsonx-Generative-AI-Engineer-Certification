\section{Introduction to Prompt Engineering}

Prompt engineering is the art and science of crafting effective inputs to guide LLMs toward desired outputs. It's one of the most critical skills for working with generative AI.

\begin{keybox}
A \textbf{prompt} is the input text provided to an LLM that instructs it on what task to perform. Effective prompts combine clear instructions, relevant context, examples, and constraints.
\end{keybox}

\section{Key Takeaways}

\begin{itemize}[leftmargin=*]
    \item Zero-shot prompting relies on model's training; few-shot provides examples
    \item Effective prompts have clear instructions, relevant context, and examples
    \item Temperature controls randomness: low (0.1-0.3) for factual, high (0.7-1.0) for creative
    \item IBM Prompt Lab provides an intuitive interface for experimentation
    \item Always test prompts with edge cases and diverse inputs
    \item Watch for prompt injection and jailbreaking attempts in production
\end{itemize}
